Behind the Scenes: The Technology at Facebook

The Front End

Facebook uses many different programs, tools, and services to make up the technology stack that it utilizes to run the site. For the front end of the site, the Facebook servers are running on a stack with Memcache which is comprised of a LAMP. A LAMP stands for Linux, Apache, MySQL, and PHP. Linux is an open source computer operating system kernel sort of like Unix. In addition to being open source, Linux is customizable and great for security. Facebook use Apache HTTP servers to run the Linux operating system.

        MySQL is also used when it comes to Facebook. Facebook uses MySQL for its gigantic database. MySQL is mostly used as a key-value store as data is randomly distributed amongst a large set of logical instances.(CITE) MySQL is a good choice because it is speedy and reliable.

        As far as programming language goes, Facebook uses PHP because it has a lot of support and a very active community and handles heavy change well. Memcache is used as a system for memory caching. This caches data in RAM so that it reduces reading time. It also helps to relieve some of the load on the database.

The Back End

        Facebook uses a variety of different programming languages to make up its back end. C++, Java, Python, and Erlang are the different languages that are used. The way that they go about creating services is done in a three step process. The first step is creating a services if it is needed. The second step is to create a framework/toolset for easier creation of services. And the third step is to use the right programming language for the task. Thrift is a protocol that Facebook developed. It basically is a cross-language services development tool. It is quick and helps save development time overall. Scribe is a log server that is used to collect and then stream data in real-time from a bunch of other servers. Cassandra is another database management system that Facebook uses to help handle large amounts of data that are spread out over multiple servers.

Storage

        Facebook has an extremely large amount of users all uploading multiple amounts of pictures. There are over 200 billions photos stored on Facebook. Facebook use 7 petabytes of storage every month. Just recently Facebook has built a new data center that is about 62,000 square feet. There is a total of 1 exabyte of storage there made up of 500 racks holding 2 petabytes of data.
